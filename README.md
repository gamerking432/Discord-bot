# ✨ Shody Vai
Your Server’s Smart Guardian & DJ

Shody Vai is a Discord bot that helps keep your server safe, organized, and fun with moderation tools and music features.

---

## 🚀 Features

### 🛡 Moderation
- Detects inappropriate language  
- Warns users automatically  

### 🎵 Music
- Play songs in voice channels  
- Search music using keywords  

### ⚡ Utilities
- Role management  
- Server tools  

---

## 📜 Commands

### 👤 Users

| Command | Description |
|--------|------------|
| !join | Join voice channel |
| !leave | Leave voice channel |
| !play <song> | Play music |
| !stop | Stop music |
| !secret | Check special access |

---

### 🛡 Moderators

| Command | Description |
|--------|------------|
| !assign @user @role | Give role |
| !rmv @user @role | Remove role |

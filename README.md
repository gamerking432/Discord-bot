✨ Shody Vai — Command Guide
👤 User Commands (Everyone Can Use)
Command
Description
!ping (if added)
Checks if the bot is alive and responsive
!hello (if added)
Greets the user
!join
Makes the bot join your current voice channel
!leave
Makes the bot leave the voice channel
!play <song name>
Plays music from YouTube based on your search
!stop
Stops the currently playing music
!secret
Checks if you belong to the special Friends role
